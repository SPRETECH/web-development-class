How to Host your website on GitHub Pages

GitHub.com -- click here

Create Account

Create new repository - make sure there is no spacing between the name of the repository.

Next step is:

Look for that repository and open that.

Look for CODE button in that

Click on that and copy the URL. (example: https://github.com/SPRETECH/web-development-class.git )

Next step - Clone project in your computer.

open terminal anywhere in your computer. (example: powershell, gitbash or any other.)

Type the following command in terminal:

git clone https://github.com/SPRETECH/web-development-class.git Do a bit change in that URL: git clone https://USERNAME:TOKEN@github.com/SPRETECH/web-development-class.git

How to get the TOKEN:

Click on Profile/Account Icon on top right corner of the Github, and click Settings.

New Page will Open

NOW: Look for the Developer Settings on the left Navigation bar (Scroll and you'll see it in the bottom of the menu on left.)

Now click Personal Token -> classic on the left side of the menu.

You'll see a button Create new token -- click that.

Create new (Classics)

In the note mention any name - without spacing.

Select Checkbox - repo only

Now scroll and click button generate token.

WARNING! - Token will be apprearing the very first time only. It'll disappear after that. So copy that and save it somewhere safe.
If you forget to copy that, you'll have to generate new token again.

NOW LET'S Connect our computer to the GitHub repository

In your terminal, type the following command:

Example below:

git clone https://USERNAME:TOKEN@github.com/SPRETECH/web-development-class.git

It'll not create a copy of that project in your Computer.

FEEL FREE TO DO EDITS IN YOUR PROJECT DIRECTORY

git status -> it will check which files are changed. git add . -> it'll add all the changed files to the staging area. git commit -m "Add initial commit" -> adding message to the commit. git push origin main -> pushing the changes to the GitHub repository.
